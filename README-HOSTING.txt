Hosting-ready portfolio package

Upload the contents of this folder to your hosting public folder.

Static pages included:
- index.html
- todo-app/
- my-first-project/
- my-second-project/

PHP pages included:
- php-projects/crud/
- php-projects/authentication/

For PHP CRUD and authentication to work, use hosting with PHP and MySQL support.
Update the database credentials in:
- php-projects/crud/connection.php
- php-projects/authentication/config.php

If you use only static hosting such as GitHub Pages or Netlify static hosting,
the HTML/CSS/JavaScript websites will work, but PHP and MySQL pages will not run.
